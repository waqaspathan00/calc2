What Is Pylint And How To Implement It

Pylint is a tool that reads through your code and helps you maintain a universal format, PEP8, that makes it more readable. This can entail adding docstrings to describe a block of code (like functions and classes), and also using proper spacing so the code is not congested

Step 1: Installing pylint. In the terminal enter "pip install pylint"

Step 2: A basic function not conforming to PEP8 standards.

Step 3: Now to lint our program. In terminal, enter "pylint {filename}". Our class may work, but it's not very readable. Let's now implement the requirements suggested by Pylint
