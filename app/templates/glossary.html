Glossary of Programming Terms

Instantiation

Creating an object

Object

An instance of a class

Class

A collection of related attributes and behaviors

Namespace

Defining a section of a program where specific classes and functions are located

Constructor

A function called upon object instantiation. Sets up the core attributes of a given class

Fixture

A fixture acts as the Arrange steps in AAA testing. Fixtures are useful for condensing pytests that have repeated behaviors

Unit Test

Testing the code to see if functions and attributes are behaving as intended
